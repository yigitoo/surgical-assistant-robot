Service Name Method Usage
Get Actuator's All Sensor Values By ID GET /id/<int:id>
Send a cmd to motors/actuators. POST / 
_______________________________________________________________

  P.S: 
    sensor_name:
      * current
      * temperature
      * angular_info
      * cartesian_info
      * torque_value
    In post request:
      cmd_name:
        * admittance_mod
        * set_cartesian (end-effector)
        * set_angular  
      cmd_val:
        * certain data types like
          1. motor_id
          2. motor_action
          3. wanted_infos like in <sensor_name>
    POST requests need to be in json forat.
